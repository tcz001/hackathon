# Crypedit

## Prerequisite

- `yarn install` 

## Built in tasks:

- `yarn chain`: starting in-memory blockchain with `ganache-cli`
- `yarn test`: run contract's test in `./test`
- `yarn deploy`: deploy compiled code to blockchain
- `yarn start`: run frontend react project
